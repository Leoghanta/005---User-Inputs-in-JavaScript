Workshop 5: Working with User Inputs in JavaScript

Workshop 5

Welcome to Workshop 5! Today, we'll dive into one of the essential aspects of JavaScript: working with user inputs. User interaction is a critical component of any dynamic web application, and JavaScript provides several tools to gather and display information. In this session, we'll focus on using the prompt() function to gather user input and the alert() and console.log() functions to display output. By the end of this workshop, you should be comfortable using these functions to create simple interactive scripts.

Introduction to User Inputs in JavaScript

User inputs allow web applications to become interactive and responsive to the needs and actions of the user. JavaScript provides several built-in methods to handle user inputs and outputs. The most basic ones we'll discuss today are:

prompt(): This function prompts the user to input some data.

alert(): This function displays an alert box with a message.

console.log(): This function logs a message to the browser's console.

These functions are fundamental to understanding how JavaScript interacts with users, so let's start with the basics.

Using prompt() to Gather User Input

The prompt() function is a simple way to ask the user to provide input. It displays a dialog box with an optional message, and the user can type their response in a text field.

Syntax of 'prompt()'


let userInput = prompt("Please enter your name:");

In this example, the prompt() function displays a dialog box with the message "Please enter your name:". Whatever the user types into the box is then stored in the variable userInput.

Example: Asking for User Input


let age = prompt("How old are you?");
console.log("You are " + age + " years old.");

This script asks the user for their age and then logs the response to the console. Notice how the prompt() function captures the user's input as a string.

Handling Cancelation or Empty Inputs

If the user clicks "Cancel" or leaves the input blank, prompt() returns null or an empty string. It's important to check for these cases to prevent unexpected behavior. 


let age = prompt("How old are you?");
if (age === null || age === "") {
    console.log("User did not enter their age.");
} else {
    console.log("You are " + age + " years old.");
}

Displaying Output with alert() and console.log()

After gathering input from the user, you often want to display it back to them or log it for debugging purposes.

The alert() Function

The alert() function displays a simple pop-up message to the user. This can be useful for quick notifications or to provide feedback based on user input.


alert("Hello, world!");

This code snippet will display a dialog box with the message "Hello, world!". The user must click "OK" to dismiss the alert and continue.

Example: Displaying a Greeting


let name = prompt("What is your name?");
alert("Hello, " + name + "!");

In this example, the user's name is requested through prompt(), and an alert is shown with a personalized greeting.

The console.log() Function

While alert() is useful for user-facing messages, console.log() is essential for developers to output messages to the browser's console. This is especially helpful for debugging purposes.


console.log("This is a debug message.");

Example: Logging Input for Debugging


let name = prompt("What is your name?");
console.log("User entered: " + name);
alert("Hello, " + name + "!");

Here, the user's input is both displayed in an alert and logged in the console, which allows the developer to see what the user entered without disrupting the user's experience.

Combining prompt(), alert(), and console.log() for Interactive Scripts

Now that we've covered the basics, let's combine these functions to create a more interactive experience.

Example: Simple Calculator


let num1 = prompt("Enter the first number:");
let num2 = prompt("Enter the second number:");

let sum = Number(num1) + Number(num2);

alert("The sum of " + num1 + " and " + num2 + " is " + sum);
console.log("Sum calculated: " + sum);

This script asks the user to input two numbers, adds them together, and displays the result in both an alert and the console. Note that Number() is used to convert the input strings to numbers before performing the addition.

Example: User Authentication Simulation


let username = prompt("Enter your username:");
let password = prompt("Enter your password:");

if (username === "admin" && password === "1234") {
    alert("Login successful!");
    console.log("User " + username + " logged in successfully.");
} else {
    alert("Login failed. Please check your username and password.");
    console.log("Login attempt failed for user: " + username);
}

In this script, the user is asked to enter a username and password. The input is checked against predefined credentials, and the result is displayed using both alert() and console.log().
